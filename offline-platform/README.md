# 🚀 Offline Programming Learning Platform

A complete, offline web-based programming learning platform with interactive courses, quizzes, and a code compiler. Works entirely in your browser with no internet connection required.

## 📋 Project Structure

```
offline-platform/
├── index.html              # Dashboard - Main entry point
├── html-course.html        # HTML course page
├── css-course.html         # CSS course page
├── js-course.html          # JavaScript course page
├── python-course.html      # Python course page
├── c-course.html           # C Programming course page
├── quiz.html               # Quiz page
├── compiler.html           # Code compiler/executor
├── css/
│   └── style.css           # All styling (modern, responsive UI)
└── js/
    ├── navigation.js       # Navigation and page routing
    ├── quiz.js            # Quiz system with scoring
    └── compiler.js        # Code execution engine
```

## 🚀 Getting Started

### Prerequisites
- Firefox browser (or any modern web browser)
- No internet connection needed!
- No installation required!

### How to Run

1. **Navigate to the project folder:**
   ```bash
   cd offline-platform
   ```

2. **Open in Firefox:**
   ```bash
   firefox index.html
   ```
   
   Or simply double-click `index.html` to open it in your default browser.

3. **That's it!** The platform is now running completely offline.

## 🧪 Quiz System

- **5 multiple-choice questions** per course
- **Score calculation** - Shows immediate feedback
- **Correct answer highlighting** - Learn from mistakes
- **Explanations** - Educational feedback for wrong answers
- **No time limits** - Learn at your own pace

### How to Take a Quiz
1. From the dashboard, click "Take Quiz" on any course card
2. Answer all 5 questions
3. Click "Submit Quiz"
4. See your score and review explanations
5. Click "Retake Quiz" to try again

## 💻 Code Compiler

A fully functional offline code editor and executor supporting:

### Supported Languages
1. **HTML** - Live preview in iframe
2. **CSS** - Styled preview rendering
3. **JavaScript** - Full execution with console.log support
4. **Python** - Code visualization and analysis
5. **C Programming** - Code visualization and analysis

### Features
- **Live Preview** - See HTML/CSS changes instantly
- **Code Execution** - Run JavaScript code safely
- **Error Handling** - Clear error messages
- **Output Console** - See program output
- **Example Code** - Pre-loaded code examples for each language
- **Safe Execution** - All code runs in browser, completely offline

### How to Use
1. Click "Open Compiler" from the dashboard
2. Select a programming language
3. Write or modify code in the editor
4. Click "▶ Run Code" to execute
5. See output in the right panel
