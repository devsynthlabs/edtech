What is CSS?

CSS stands for Cascading Style Sheets. CSS is used to style and layout web pages and to add effects and animations to make websites more visually appealing and interactive.

While HTML provides the structure and content, CSS controls the presentation. CSS works with HTML to create beautiful, responsive web designs.

The Three Ways to Add CSS:

  • Inline CSS: Added directly to HTML elements
  • Internal CSS: Added in <style> tags in the HTML head
  • External CSS: Added via external .css files (best practice)

CSS Syntax

CSS consists of selectors and declarations. A declaration contains a property and a value.

/* Selector and Property-Value Pairs */ h1 { color: blue; /* Property: Value */ font-size: 2rem; text-align: center; }

Structure Breakdown:

  • h1 - The selector (targets HTML elements)
  • color: blue; - A declaration (property: value)
  • {} - Curly braces contain all declarations

CSS Selectors

Selectors are patterns used to select HTML elements to style. Here are the most common types:

Element Selector

p { color: red; }

Class Selector

.highlight { background-color: yellow; }

ID Selector

#header { background-color: navy; }

Universal Selector

* { margin: 0; padding: 0; }

Attribute Selector

input[type="text"] { border: 1px solid gray; }

Combinators

/* Descendant selector */ div p { color: blue; } /* Child selector */ div > p { color: red; } /* Adjacent sibling selector */ h2 + p { margin-top: 0; } /* General sibling selector */ h2 ~ p { color: green; }

Colors and Backgrounds

CSS provides multiple ways to define colors and backgrounds:

Color Properties

/* Text Color */ h1 { color: red; } /* Background Color */ body { background-color: #f0f0f0; } /* Different Color Formats */ p { color: red; /* Color name */ } h2 { color: #FF5733; /* Hex code */ } h3 { color: rgb(255, 87, 51); /* RGB */ } h4 { color: rgba(255, 87, 51, 0.5); /* RGBA with transparency */ }

Background Properties

/* Background Image */ div { background-image: url('image.jpg'); background-size: cover; background-position: center; background-repeat: no-repeat; } /* Gradient Background */ header { background: linear-gradient(135deg, #667eea, #764ba2); }

Text Styling

CSS provides many properties to style text:

p { /* Font Properties */ font-family: Arial, sans-serif; font-size: 1.2rem; font-weight: bold; font-style: italic; /* Text Properties */ color: #333; text-align: center; text-decoration: underline; text-transform: uppercase; line-height: 1.6; letter-spacing: 2px; }

The Box Model

Every element in CSS is composed of four layers: Content, Padding, Border, and Margin.

div { /* Content - Inside the element */ width: 300px; height: 200px; /* Padding - Space inside the border */ padding: 20px; /* Or individual sides */ padding-top: 10px; padding-right: 15px; padding-bottom: 10px; padding-left: 15px; /* Border - Around the padding */ border: 2px solid black; /* Margin - Space outside the border */ margin: 20px; /* Or individual sides */ margin-top: 10px; margin-bottom: 10px; }

Box Model Order (Outside to Inside):

  1. Margin - Transparent space outside
  2. Border - Line around the padding
  3. Padding - Space inside the border
  4. Content - The actual content

Display and Layout

The display property determines how an element is rendered:

/* Display Types */ .inline { display: inline; } /* Flows with text */ .block { display: block; } /* Takes full width */ .inline-block { display: inline-block; } /* Both inline and block */ .none { display: none; } /* Hidden */ /* Flexbox Layout */ .container { display: flex; justify-content: center; align-items: center; gap: 20px; } /* Grid Layout */ .grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; }

Responsive Design with Media Queries

Media queries allow you to apply different styles for different screen sizes:

/* Desktop styles */ .container { width: 1200px; display: grid; grid-template-columns: repeat(3, 1fr); } /* Tablet screens */ @media (max-width: 768px) { .container { width: 100%; grid-template-columns: repeat(2, 1fr); } } /* Mobile screens */ @media (max-width: 480px) { .container { grid-template-columns: 1fr; padding: 10px; } }

Transitions and Hover Effects

Make your website interactive with transitions and hover effects:

/* Smooth Transition */ button { background-color: blue; transition: background-color 0.3s ease; } button:hover { background-color: darkblue; transform: scale(1.05); } /* Animation */ @keyframes slideIn { from { opacity: 0; transform: translateX(-50px); } to { opacity: 1; transform: translateX(0); } } .animated { animation: slideIn 0.5s ease-in-out; }

Most Common CSS Properties

width, height - Element dimensions background-color - Background color color - Text color border - Border around element margin - Space outside element padding - Space inside element font-size - Text size font-family - Font type display - How element is displayed position - Element positioning z-index - Stacking order opacity - Element transparency box-shadow - Shadow effect border-radius - Rounded corners transform - Scale, rotate, skew elements
