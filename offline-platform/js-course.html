Learn JavaScript - The programming language of the web

What is JavaScript?

JavaScript is a powerful programming language that runs in web browsers. It allows you to add interactivity and dynamic behavior to web pages.

JavaScript can:

  • Update HTML content dynamically
  • Respond to user events (clicks, typing, etc.)
  • Validate form data
  • Create animations and visual effects
  • Store and retrieve data locally

Variables and Data Types

Variables store data values. JavaScript has several ways to declare variables: var, let, and const.

Declaring Variables

// Using var (older way, not recommended) var name = "John"; // Using let (preferred for variables that change) let age = 25; // Using const (preferred for variables that don't change) const PI = 3.14159;

Data Types

// String let message = "Hello, World!"; // Number let count = 42; let price = 19.99; // Boolean let isActive = true; let isComplete = false; // Array let colors = ["red", "green", "blue"]; // Object let person = { name: "John", age: 30, city: "New York" }; // Null and Undefined let empty = null; let notDefined = undefined;

Basic Operations

JavaScript supports various operations on data:

Arithmetic Operations

let a = 10; let b = 3; console.log(a + b); // 13 (addition) console.log(a - b); // 7 (subtraction) console.log(a * b); // 30 (multiplication) console.log(a / b); // 3.33 (division) console.log(a % b); // 1 (modulo - remainder) console.log(a ** b); // 1000 (exponentiation)

String Operations

let firstName = "John"; let lastName = "Doe"; // Concatenation let fullName = firstName + " " + lastName; // Template Literals let message = `Hello, ${firstName}! You are learning JavaScript.`; // String Methods let text = "JavaScript"; console.log(text.length); // 10 console.log(text.toUpperCase()); // JAVASCRIPT console.log(text.toLowerCase()); // javascript console.log(text.charAt(0)); // J console.log(text.substring(0, 4)); // Java

Conditional Statements

Conditionals allow your code to make decisions based on conditions:

if Statement

let age = 18; if (age >= 18) { console.log("You are an adult"); } else { console.log("You are a minor"); }

if...else if...else Statement

let score = 85; if (score >= 90) { console.log("Grade: A"); } else if (score >= 80) { console.log("Grade: B"); } else if (score >= 70) { console.log("Grade: C"); } else { console.log("Grade: F"); }

switch Statement

let day = 3; let dayName; switch (day) { case 1: dayName = "Monday"; break; case 2: dayName = "Tuesday"; break; case 3: dayName = "Wednesday"; break; default: dayName = "Unknown"; } console.log(dayName); // Wednesday

Comparison Operators

=== / !== : Strictly equal / not equal == / != : Loosely equal / not equal > / < : Greater than / Less than >= / <= : Greater than or equal / Less than or equal && : AND operator || : OR operator ! : NOT operator

Loops

Loops repeat a block of code multiple times:

for Loop

// Print numbers 1 to 5 for (let i = 1; i <= 5; i++) { console.log(i); }

while Loop

let i = 1; while (i <= 5) { console.log(i); i++; }

do...while Loop

let i = 1; do { console.log(i); i++; } while (i <= 5); // Runs at least once, even if condition is false

for...of Loop (for Arrays)

let fruits = ["apple", "banana", "orange"]; for (let fruit of fruits) { console.log(fruit); }

forEach Method

let numbers = [1, 2, 3, 4, 5]; numbers.forEach(function(num) { console.log(num); });

Functions

Functions are reusable blocks of code that perform specific tasks:

Function Declaration

// Define a function function greet(name) { return "Hello, " + name + "!"; } // Call the function console.log(greet("John")); // Hello, John!

Arrow Functions

// Modern syntax const add = (a, b) => { return a + b; }; // Short syntax const multiply = (a, b) => a * b; console.log(add(5, 3)); // 8 console.log(multiply(4, 5)); // 20

Function with Default Parameters

function greet(name = "Guest") { console.log("Welcome, " + name); } greet(); // Welcome, Guest greet("Alice"); // Welcome, Alice

Objects and Arrays

Complex data structures to organize and store data:

Objects

let student = { name: "John", age: 20, grade: "A", subjects: ["Math", "English", "Science"], // Methods greet: function() { return "Hi, I'm " + this.name; } }; // Accessing properties console.log(student.name); // John console.log(student["age"]); // 20 console.log(student.greet()); // Hi, I'm John

Arrays and Methods

let numbers = [1, 2, 3, 4, 5]; // Array methods console.log(numbers.length); // 5 console.log(numbers[0]); // 1 numbers.push(6); // Add to end numbers.pop(); // Remove from end numbers.unshift(0); // Add to beginning let sliced = numbers.slice(1, 3); // Get elements 1-2 // Array iteration let doubled = numbers.map(n => n * 2); let evens = numbers.filter(n => n % 2 === 0); let sum = numbers.reduce((a, b) => a + b, 0);

DOM Manipulation

The DOM (Document Object Model) allows JavaScript to interact with HTML elements:

// Get elements let element = document.getElementById("myId"); let elements = document.getElementsByClassName("myClass"); let elements2 = document.querySelectorAll(".myClass"); // Change content element.textContent = "New text"; element.innerHTML = "<p>New HTML</p>"; // Change attributes element.setAttribute("class", "newClass"); element.id = "newId"; // Change styles element.style.color = "red"; element.style.fontSize = "20px"; // Add event listeners element.addEventListener("click", function() { console.log("Element clicked!"); });

Event Handling

Respond to user interactions:

// Mouse Events element.addEventListener("click", handler); element.addEventListener("mouseover", handler); element.addEventListener("mouseout", handler); element.addEventListener("mousemove", handler); // Keyboard Events element.addEventListener("keydown", handler); element.addEventListener("keyup", handler); element.addEventListener("keypress", handler); // Form Events element.addEventListener("submit", handler); element.addEventListener("change", handler); element.addEventListener("input", handler); // Window Events window.addEventListener("load", handler); window.addEventListener("resize", handler); window.addEventListener("scroll", handler);

Modern JavaScript (ES6+)

Modern JavaScript features make code cleaner and more powerful:

Destructuring

// Array destructuring const [first, second] = [1, 2, 3]; // Object destructuring const { name, age } = { name: "John", age: 30 }; // Function parameters const greet = ({ name, age }) => { return `${name} is ${age} years old`; };

Spread Operator

let arr1 = [1, 2, 3]; let arr2 = [4, 5, 6]; // Combine arrays let combined = [...arr1, ...arr2]; // Copy array let copy = [...arr1]; // Spread in function call Math.max(...arr1); // 3
