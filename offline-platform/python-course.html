Learn Python - A beginner-friendly and powerful programming language

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It's one of the most popular languages for beginners and professionals alike.

Why Python?

  • Easy to learn and understand
  • Large standard library with many built-in functions
  • Used in web development, data science, AI, and automation
  • Community support and extensive documentation
  • Indentation makes code readable

Variables and Data Types

Python uses dynamic typing - you don't need to declare variable types explicitly.

# String name = "Alice" message = 'Hello, World!' # Integer age = 25 count = -5 # Float (Decimal) price = 19.99 temperature = 36.5 # Boolean is_active = True is_complete = False # List (Array) colors = ["red", "green", "blue"] numbers = [1, 2, 3, 4, 5] # Tuple (Immutable List) coordinates = (10, 20) # Dictionary (Key-Value pairs) person = { "name": "John", "age": 30, "city": "New York" } # Set (Unique values) unique_numbers = {1, 2, 3, 4, 5}

Type Checking

x = 10 print(type(x)) # <class 'int'> y = "Hello" print(type(y)) # <class 'str'> z = [1, 2, 3] print(type(z)) # <class 'list'>

The Print Function

The print() function displays output to the console:

# Basic print print("Hello, World!") # Print multiple items print("Name:", "John", "Age:", 30) # String concatenation name = "Alice" print("Welcome, " + name) # String formatting print(f"My name is {name} and I am {25} years old") # Print with different separators print("a", "b", "c", sep="-") # a-b-c # Print with newline handling print("Line 1", end=" ") print("Line 2") # Line 1 Line 2

Getting User Input

The input() function gets user input from the keyboard:

# Get string input name = input("Enter your name: ") print(f"Hello, {name}!") # Get numeric input age = int(input("Enter your age: ")) print(f"You are {age} years old") # Get float input price = float(input("Enter price: ")) print(f"Price: ${price}") # Multiple inputs x, y = input("Enter two numbers (space-separated): ").split() print(f"x = {x}, y = {y}")

Operators

Python supports various operators for mathematical and logical operations:

Arithmetic Operators

a = 10 b = 3 print(a + b) # 13 (addition) print(a - b) # 7 (subtraction) print(a * b) # 30 (multiplication) print(a / b) # 3.33... (division) print(a // b) # 3 (floor division) print(a % b) # 1 (modulo) print(a ** b) # 1000 (exponentiation)

Assignment Operators

x = 10 x += 5 # x = 15 x -= 3 # x = 12 x *= 2 # x = 24 x /= 4 # x = 6 x //= 2 # x = 3

Comparison Operators

a = 5 b = 10 print(a == b) # False (equal) print(a != b) # True (not equal) print(a < b) # True (less than) print(a > b) # False (greater than) print(a <= b) # True (less or equal) print(a >= b) # False (greater or equal)

Logical Operators

x = True y = False print(x and y) # False print(x or y) # True print(not x) # False

Conditional Statements

Control program flow based on conditions:

if Statement

age = 18 if age >= 18: print("You are an adult") else: print("You are a minor")

if...elif...else Statement

score = 85 if score >= 90: print("Grade: A") elif score >= 80: print("Grade: B") elif score >= 70: print("Grade: C") else: print("Grade: F")

Nested if

age = 25 income = 50000 if age >= 18: if income >= 30000: print("You qualify for the loan") else: print("Income is too low") else: print("You must be 18 or older")

Loops

Repeat code blocks efficiently:

for Loop

# Loop through range for i in range(1, 6): print(i) # Loop through list fruits = ["apple", "banana", "orange"] for fruit in fruits: print(fruit) # Loop with index for i, fruit in enumerate(fruits): print(f"{i}: {fruit}")

while Loop

i = 1 while i <= 5: print(i) i += 1 # Loop with break count = 0 while True: print(count) count += 1 if count >= 5: break # Exit loop

Loop Control

for i in range(1, 6): if i == 3: continue # Skip this iteration if i == 5: break # Exit loop print(i)

Functions

Reusable blocks of code:

Function Definition

# Basic function def greet(): print("Hello!") greet() # Call the function # Function with parameters def greet_person(name): print(f"Hello, {name}!") greet_person("Alice") # Function with return value def add(a, b): return a + b result = add(5, 3) print(result) # 8

Function with Default Parameters

def greet(name="Guest"): print(f"Welcome, {name}!") greet() # Welcome, Guest! greet("Alice") # Welcome, Alice!

Multiple Return Values

def get_min_max(numbers): return min(numbers), max(numbers) minimum, maximum = get_min_max([3, 1, 4, 1, 5]) print(f"Min: {minimum}, Max: {maximum}")

Lists and Dictionaries

Important data structures in Python:

List Operations

numbers = [1, 2, 3, 4, 5] # List methods numbers.append(6) # Add to end numbers.insert(0, 0) # Insert at position numbers.remove(3) # Remove value popped = numbers.pop() # Remove and return last # List slicing print(numbers[1:4]) # Elements 1-3 print(numbers[:3]) # First 3 elements print(numbers[2:]) # From element 2 to end # List comprehension squares = [x**2 for x in range(1, 6)] # [1, 4, 9, 16, 25]

Dictionary Operations

person = { "name": "John", "age": 30, "city": "New York" } # Access values print(person["name"]) # John print(person.get("age")) # 30 # Add/Update person["email"] = "john@example.com" person["age"] = 31 # Delete del person["city"] # Iterate for key, value in person.items(): print(f"{key}: {value}")

String Operations

Working with strings in Python:

text = "Hello, World!" # String methods print(text.upper()) # HELLO, WORLD! print(text.lower()) # hello, world! print(text.replace("World", "Python")) print(text.split(",")) # Split by comma print(len(text)) # 13 # String indexing print(text[0]) # H print(text[-1]) # ! print(text[0:5]) # Hello # String formatting name = "Alice" age = 25 print(f"My name is {name} and I am {age} years old") print("Name: {}, Age: {}".format(name, age))
